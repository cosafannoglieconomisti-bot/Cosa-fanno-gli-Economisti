# FR - Les profs sont-ils racistes ? Révéler les stéréotypes

## Description
L'étude 'Revealing Stereotypes: Evidence from Immigrants in Schools' de Carlana, publiée dans l'American Economic Journal: Economic Policy en 2019, demande : Le biais inconscient d'un enseignant peut-il changer l'avenir de ses élèves ? Cette recherche utilise des tests d'association implicite pour révéler comment les stéréotypes sur les immigrés affectent les performances scolaires et les choix des jeunes en Italie.
